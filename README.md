Описание проекта
Анализ покупательского поведения пользователей маркетплейса OZON в категории «Одежда, обувь, аксессуары» без построения предиктивных моделей.
Цель проекта: Комплексный анализ покупательского поведения пользователей с фокусом на статистических и исследовательских задачах, включая анализ сезонности, сегментацию категорий, географическое распределение, метрики удержания пользователей и т.д.
Источник данных
Данные взяты из соревнования по машинному обучению Ozon 2025.
Официальный источник:
Соревнование на Kaggle: Ozon 2025 ML Competition
Данные для проекта: https://www.kaggle.com/datasets/thunderpede/testststs/data
